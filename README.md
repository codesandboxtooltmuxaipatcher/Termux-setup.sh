# termux-tools-kali-ubuntu
Installation of various operating systems such as Ubuntu, Kali Linux and Debian


install :

```
pkg update
```
```
pkg upgrade
```
```
pkg install git
```
```
git clone https://github.com/salehbngpy/termux-tools-kali-ubuntu.git
```
```
cd termux-tools-kali-ubuntu
```
```
chmod +x BNG.sh
```
```
bash BNG.sh
```
